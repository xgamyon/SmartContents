float

1. 공중에 붕 뜨다
2. 플룻이 선언된 엘리먼트는 컨텐츠가 피해간다.
3. 플룻이 선언된 엘리먼트는 먼저 작성된 엘리먼트보다 위로 뜰수 없다!

width: auto
height: auto



주어진 영역을 벗어났을때 부모만큼으로 맞춰라!! (마스크 개념)

overflow: hidden  넘어간 영역을 숨겨라.
              visible   주어진 영역을 벗어날수 있다!(디폴트 상태)
              scroll    넘어간 영역을 숨기되 스크롤을 생기게 하라.





outline  -> border는 px 수치로 영향을 주지만 outline은 영향을 주지 않는다!
overflow:hidden  -> 부모가 자식 사이즈 만큼 가려주는것!! (자식 사이즈만큼 부모사이즈를 감싸준다.)


.contents .left   -> 컨텐츠 안에 있는 모든 레프트 들에게..
.contents> .left   -> 컨텐츠 안에 있는 바로 밑의 레프트에게..

class 같은 경우는 두가지의 css 도 먹일수 있다!
예 ) <div class="a1 a2">


height 값은 부모가 항상 값이 존재해야 한다.!!!
